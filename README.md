# Zenith-Habits

## Primary Goal

The primary goal of Zenith Habits is to simplify the process of tracking and maintaining daily responsibilities and habits. With intuitive tools to add and monitor habits, the app provides detailed statistics to visually demonstrate your consistency and progress. Additionally, the integrated journal feature allows you to reflect on your daily experiences, giving you deeper insights into your personal growth and habit development. Zenith Habits isn't just about tracking; it's about transforming your daily routines into stepping stones for success.

## To run

```bash
$ flutter pub get

$ flutter run
```
